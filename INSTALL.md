# Instruções de Instalação

## Pré-requisitos
- Android 5.0+ (API 21+)
- Mínimo 500MB de espaço livre
- Câmera funcional

## Instalação via ADB
1. Conecte seu celular via USB
2. Ative o modo de desenvolvedor
3. Execute: adb install AutoAimDetector-Android.apk

## Instalação Manual
1. Copie o arquivo .apk para seu celular
2. Toque no arquivo para instalar
3. Conceda permissões de câmera

## Permissões Necessárias
- Câmera
- Internet (para atualizar modelo YOLO)
- Acesso à rede

## Solução de Problemas
Se o app não inicia:
- Reinicie o celular
- Limpe o cache do app
- Reinstale o aplicativo
